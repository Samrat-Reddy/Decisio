Project Title: Decisio – A Rule-Based Workflow & Decision Automation System

Abstract:
Decisio is a web-based application designed to streamline and automate multi-level approval processes within a college environment. The system facilitates structured handling of requests such as student leave applications, faculty leave, budget approvals, and event permissions by implementing a dynamic, rule-based workflow engine.

Unlike traditional approval systems that follow fixed hierarchical paths, Decisio introduces a configurable decision framework where approval chains are generated dynamically based on predefined conditions such as request type, duration, priority, and financial thresholds. This ensures that each request is routed only through the necessary authorities, reducing delays and improving operational efficiency.

The system supports multiple user roles including Student, Mentor, Faculty, Head of Department (HOD), Principal, Dean, and Administrator. Each role has clearly defined permissions, enabling controlled access and preventing unauthorized actions such as self-approval. Students can submit requests, mentors and faculty act as initial reviewers, while higher authorities such as HOD, Principal, and Dean are involved conditionally based on the complexity and importance of the request.

Decisio also maintains a complete audit trail of all decisions, including timestamps, approver details, and comments, ensuring transparency and accountability. Additional features include real-time status tracking, notifications, document upload and management, and an administrative panel for configuring workflows and managing users.

Built using modern web technologies such as React.js for the frontend and Node.js with Express for the backend, the system follows RESTful API design principles and supports secure authentication using JWT. Data persistence is handled through a relational database, enabling efficient storage and retrieval of structured information.

By combining rule-based decision logic with modular system design, Decisio not only fulfills the requirements of a standard academic project but also demonstrates real-world applicability in automating institutional workflows and improving decision-making efficiency.
